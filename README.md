# Geo-Based Asset Tracking System

A full-stack MERN application for tracking assets on a map with role-based authentication.

## Tech Stack
- **Frontend**: React, Vite, TailwindCSS, Leaflet
- **Backend**: Node.js, Express, MongoDB
- **Database**: MongoDB (GeoJSON)

## Setup

### Prerequisites
- Node.js
- MongoDB

### Installation
1. Clone the repo.
2. Install dependencies in `server` and `client`.
   ```bash
   cd server && npm install
   cd ../client && npm install
   ```
3. Run the development servers.
